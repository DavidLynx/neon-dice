About Neon Dice

Neon Dice es una web app RPG bilingüe creada como herramienta funcional y pieza de portafolio interactiva. Combina perfiles locales, dados animados, HP Tracker, presets, encuentros e historial exportable en una experiencia estática construida con HTML, CSS y JavaScript vanilla. No usa cuentas, servidores ni APIs externas.
